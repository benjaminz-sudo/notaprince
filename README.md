# notaprince
Rust E4 project about procedural generation

This is a simple Rust minigame project that takes heavy inspiration from the game [Blue Prince](https://store.steampowered.com/app/1569580/Blue_Prince/).
The player must navigate a map blindly from room to room, and try to reach the end room.
The twist is that the rooms the player can navigate to are randomly generated using a set seed the player must input each time he moves.
Using knowledge from previous runs, the player can harness the random and have an easier time navigating to the end room.
